## Overview

The Web-Based Garbage Truck Tracking System was developed to enhance waste management operations for the Municipal Environment and Natural Resources Office (MENRO) of Balingasag. Utilizing Scikit-Learn Analytics, the system optimizes garbage truck tracking, ensuring efficient waste collection and disposal.

## Key Features

- **Real-Time Truck Tracking**: Monitors garbage truck locations to optimize collection routes.
- **Data-Driven Analytics**: Utilizes Scikit-Learn to analyze waste collection patterns and improve efficiency.
- **Automated Reporting**: Generates reports on collection performance and operational insights.
- **User-Friendly Dashboard**: Provides an intuitive interface for monitoring and managing waste collection.
- **Operational Efficiency**: Minimizes manual workload and enhances decision-making for waste management.

## Technologies Used

- **Programming Languages**: Python, PHP, JavaScript
- **Frontend Technologies**: HTML, CSS, JavaScript
- **Database Management**: MySQL
- **Machine Learning & Data Analysis**: Scikit-Learn (for waste collection pattern analysis)
- **Version Control**: Git, Github
- **Deployment & Hosting**: Apache, Hostinger
- **Tracking & Mapping**: Google Maps API, LeafletJS

## Challenges and Learnings

Developing the system presented challenges in ensuring data accuracy for real-time tracking, optimizing route efficiency, and integrating multiple system modules seamlessly. Implementing Scikit-Learn for data analysis required refining algorithms to provide meaningful insights. Additionally, designing a user-friendly dashboard was crucial to ensure accessibility for municipal staff. Overcoming these challenges enhanced my skills in machine learning integration, system optimization, and user experience design.

## Outcome

The system significantly improved waste collection efficiency by optimizing routes, reducing fuel consumption, and minimizing travel time. Through data-driven decision-making, MENRO Balingasag gained valuable insights for better waste management planning. The automated reporting feature streamlined operations, reducing manual workload and enhancing productivity. Overall, the system contributed to a more efficient, organized, and sustainable waste management process for the municipality.
