## Overview

The Mobile App for Lily's Home and Pizza Ventures Inc. – Santino’s Pizza was developed to streamline inventory management and sales reporting for crew members. The app provides an intuitive interface for tracking stock levels, and adding sales reports. Designed for efficiency, it enhances operational transparency and ensures seamless inventory control.

## Key Features

- **Crew Interface**: Allows employees to log sales reports and monitor inventory in real-time.
- **Stock Management**: Tracks available stock levels, preventing shortages and ensuring smooth operations.
- **Sales Reporting**: Enables crew members to input daily sales data efficiently.
- **User-Friendly Design**: Intuitive navigation tailored for crew members with minimal technical experience.
- **Mobile Accessibility**: Ensures on-the-go access for seamless business operations.

## Technologies Used

- **Frontend**: React Native (with TypeScript & Tailwind CSS for styling)
- **Backend**: Laravel API
- **Database**: MySQL
- **Authentication**: Laravel Sanctum
- **Version Control**: Git, Github

## Challenges and Learnings

Developing the app required ensuring seamless inventory tracking while maintaining a simple user interface for crew members. Designing an intuitive system that allows employees to log sales and monitor stock efficiently was a key challenge. Integrating real-time updates while optimizing database queries improved my understanding of efficient API design and data synchronization. Additionally, balancing performance and user experience strengthened my skills in mobile UI/UX design and backend optimization.

## Outcome

The mobile app improved inventory accuracy and sales reporting efficiency, reducing errors and enhancing transparency for the Santino’s Pizza Division. Crew members could easily track stock levels, preventing shortages and optimizing restocking processes. By leveraging data-driven insights, management gained a clearer view of sales trends, leading to better decision-making and improved operational efficiency.
